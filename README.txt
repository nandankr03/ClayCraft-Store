Project Name: Clay Pottery & Sandstone Marketplace
Developed By: Nandan, Ansh, 
Course: BCA (Batch 2023–2026)
College: [SMS College Varanasi]

Description:
This is a web-based marketplace for handmade clay and sandstone crafts.
Users can browse products, add them to the cart, and make purchases after login/signup.

Technologies Used:
- HTML
- CSS
- JavaScript
- Node.js (Backend)
- MySQL (Database)
